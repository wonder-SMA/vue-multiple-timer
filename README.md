# This is a Vue 3 + Vite timer project

## What's inside?

- [VueJS](https://vuejs.org)
- [Vite](https://vitejs.dev)

## Getting started

1. Install dependencies.
   ```bash
   npm install
   ```

2. Start dev server with hot reload at http://localhost:3000.
   ```bash
   npm run dev
   ```

## Other commands

### Build commands

```bash
npm run build
```

```bash
npm run preview
```
